## How to install

1. Clone my repositories with link below

```bash
git clone https://github.com/azir999/meatflix
```

2. migration my database
```bash
php artisan migrate
```
3. seed my seeder

```bash
php artisan db:seed
```
4. run website

```bash
php artisan serve
```
